# Animated Search Bar using HTML & CSS - CodingNepal

Um componente de caixa de busca (Search Box) moderno, minimalista e altamente performático desenvolvido exclusivamente com HTML5 e CSS3. O projeto destaca uma microinteração visual onde um botão circular compacto contendo um ícone de lupa expande-se horizontalmente de forma fluida para revelar um campo de entrada de texto completo, operando sem a necessidade de scripts JavaScript.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do contêiner de busca, campo de entrada (`<input type="text">`) e elemento do botão de acionamento.
* CSS3: Estilização completa, uso de formas circulares perfeitas, transições suaves e gerenciamento de estados visuais.
* Animações em CSS: Utilização de propriedades de transição (`transition`) otimizadas para controlar dinamicamente a largura (*width*) e a visibilidade do campo.

## Funcionalidades e Técnicas Aplicadas

* Efeito Expansível Inteligente (Expandable Toggle): O componente inicia em um estado compacto ocultando o campo de texto. Ao interagir com o elemento (utilizando técnicas como `:hover`, `:focus-within` ou inputs ocultos), a caixa se expande de forma elástica para os lados.
* Ícone Vetorial Centralizado: Inclusão de um ícone de lupa em branco puro centralizado perfeitamente sobre um botão circular vermelho de alto contraste visual.
* Design Imersivo em Modo Escuro (Dark Mode): Uso de um plano de fundo preto puro que elimina distrações periféricas, direcionando o foco do utilizador estritamente para a interação centralizada.
* Alinhamento e Centralização com Flexbox: Posicionamento perfeitamente equilibrado do componente no centro exato da viewport.
